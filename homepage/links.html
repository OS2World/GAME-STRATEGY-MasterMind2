Tools used for Mastermind/2:

This game was entirely written in C, using IBM's Visual Age C++ 3.0 for OS/2. All Images for the program were created with GIMP/2 for XFree/2 for OS/2 or at least modified with it. And finally I used the most complex tool to create the installation-program and the online-help: OS/2's advanced editor EPM :-)



Tools used for the webpages:

The screenshots were taken with PMCamera/2 version 2.10 (no webpage available) as OS/2 PM 1.3 bitmaps and then converted with GIMP/2 to gifs. I also converted them using the built in OS/2 MMPM function, but GIMP/2 created images which eat up only a third of the space that the gifs made with MMPM used. And as another plus, GIMP/2 used a color palette optimized for webbrowsers. The design of this website is based on an idea of Cheapdesign. All images on this webpage have been created or modified with GIMP/2.
