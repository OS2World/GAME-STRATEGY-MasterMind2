My "OS/2-evolution" and other projects I'm working on:

I am using OS/2 since version 2.1 and didn't miss a release :-) So my state today is Warp 4 with Fixpack 12 in german. Mastermind/2 was the first program I released to the public, but currently I am working with some other guys from around the world on project "everblue". The goal of project everblue is to port Xlib to OS/2's Presentation Manager (PM), which would enable the OS/2 community to recompile nearly every program written for the X Window system, link it against our library and run it under OS/2 natively.
